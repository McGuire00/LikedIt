# ❤️ LikedIt – See Every Post You’ve Liked

**LikedIt** is a Chrome extension that highlights every Instagram post you’ve liked on a specific user’s profile — so you can easily see all your past favorites in one view.

---

## 🚀 Features

-   💡 **Instant Highlights** – See which posts you’ve liked on any Instagram user’s page
-   ⚡ **Automatic Detection** – Works as you scroll, no clicks or setup required
-   🔒 **Privacy-First** – 100% client-side; no data is collected, stored, or sent anywhere
-   🧩 **Lightweight** – Built to blend seamlessly with Instagram’s design

---

## 🧩 How to Install (Manual Installation)

Since LikedIt isn’t published on the Chrome Web Store yet, you can install it manually in **Developer Mode**.  
Follow these steps carefully:

### 1️⃣ Download or Clone the Repository

Download this repo as a ZIP file or clone it:

```bash
git clone https://github.com/McGuire00/LikedIt
```

### 2️⃣ Open Chrome Extensions Page

In Chrome, go to:

<pre>chrome://extensions/ </pre>

### 3️⃣ Enable Developer Mode

In the top-right corner of the extensions page, toggle Developer mode to ON.

### 4️⃣ Load the Extension

Click Load unpacked, then select the folder where you extracted or cloned this repository.

After loading successfully, you’ll see the LikedIt icon appear in your Chrome toolbar. 🎉
